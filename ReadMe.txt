To run this project on your machine:

1. Clone this repository
2. Install all dependencies from Pipfule.lock (python 3.6.5+) using commands:
	cd /project_dir/	
	pipenv install
3. Run server using commands:
	cd /project_dir/
	pipenv shell
	python manage.py runserver
4. Go to localhost:8000/
